# 🌤 Weather App

## Description
A Python GUI Weather Application that fetches real-time weather information using the OpenWeatherMap API.

## Features
- Search weather by city name
- Displays:
  - Temperature (°C & °F)
  - Weather condition
  - Humidity
  - Wind speed
  - Pressure
  - Visibility
  - Sunrise & Sunset
- Weather icon
- Beautiful GUI using Tkinter
- Error handling for invalid city and network issues

## Technologies Used
- Python
- Tkinter
- Requests
- Pillow
- OpenWeatherMap API

## Installation

Install the required packages:

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python weather_app.py
```
