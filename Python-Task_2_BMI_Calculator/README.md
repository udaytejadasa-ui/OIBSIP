# BMI Calculator

## Description
A Python GUI application that calculates Body Mass Index (BMI) based on the user's height and weight.

## Features
- User-friendly GUI
- Calculates BMI instantly
- Displays BMI category
- Health tips based on BMI
- Clear and Exit buttons

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python.
2. Run:
   python bmi_calculator.py
